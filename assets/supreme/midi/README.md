# MIDI patches

This folder contains the subset of "EAWpats", a set of GUS/Timidity-compatible
sound patches for MIDI playback, needed for legacy Supreme With Cheese MIDI songs.

Obtained from:
https://www.libsdl.org/projects/old/SDL_mixer/timidity/timidity.tar.gz
2026-06-08

In turn obtained from:
http://www.stardate.bc.ca/gus_patches.htm
1999-09-27
